# 아이뱅크 안마서비스(헬스키퍼) 예약 시스템 설계서 초안

## 1. 개요

### 1.1 서비스 명칭
- **아이뱅크 안마서비스 (헬스키퍼)**

### 1.2 목적
- 사내(또는 대상자) 안마서비스의 예약을 온라인으로 신청·관리하기 위한 시스템
- 안마사 1인의 한정된 가능 시간 안에서 공정하게 예약을 배분(우선권 기반)
- 예약 신청부터 확정, 안내 메일 발송까지의 과정을 체계화

### 1.3 핵심 정책 요약
- 안마사는 **1명**이며, 안마서비스 이용은 **매일 13:30~17:00**, **타임당 30분 + 쉬는시간 30분**으로 운영된다.
- 실제 서비스 가능 슬롯: **13:30~14:00 / 14:30~15:00 / 15:30~16:00 / 16:30~17:00** (하루 4타임)
- 예약은 **주 단위**로 운영되며, **매주 수요일에만** 차주(다음 주 월~금) 예약을 신청할 수 있다.
- **매주 수요일 오전 9시 예약 오픈**, **같은 날 오후 5시 예약 마감**된다. (실제 신청은 16:59까지, 17:00 정각 마감)
- 일반 예약은 **신청(요청)** 상태로 접수되며, **관리자의 확정 처리**가 있어야 예약 완료가 된다.
- 동일 날짜·시간 중복 신청 시 **이전에 안마서비스를 받은 지 오래된 사용자에게 우선권**이 부여된다.
- 마감 후 **우선권 탈락자에 한해 다음날 오후 5시까지 비어있는 슬롯에 선착순 재신청**할 수 있으며, **재신청은 신청 즉시 예약 확정**된다. (재신청 건은 **취소 불가**)
- 일반 신청은 **마감(수 17:00) 전까지 취소 가능**, 마감 이후에는 취소 불가. (취소는 우선권에 영향 없음)
- 안마사의 **휴가는 예약 오픈 전에 미리 등록**하며, 오픈 후에는 등록 불가. 휴가 날짜는 예약 불가.

## 2. 사용자 정의 (Actor)

| 구분 | 설명 | 주요 권한 |
|------|------|-----------|
| 비회원 | 가입 전 방문자 | 회원가입, 서비스 소개 열람 |
| 회원(신청자) | 이메일 인증을 완료한 일반 사용자 | 예약 가능 날짜/시간 확인, 예약 신청·취소, 본인 예약 내역 확인, (탈락 시) 재신청 |
| 관리자 | 서비스 운영자 | 신청자 정보 확인, 예약 확정/취소, 우선권 확인, 안마사 휴가 관리, 메일 발송 |

## 3. 핵심 개념 및 비즈니스 규칙

### 3.1 안마서비스 운영 시간 / 시간 슬롯
- 안마사 인원: **1명**
- 운영 시간: 매일 **13:30 ~ 17:00**
- 단위: **타임당 30분 + 쉬는시간 30분**
- 실제 예약 가능 슬롯(하루 4타임):
  | 타임 | 시간 |
  |------|------|
  | 1타임 | 13:30 ~ 14:00 |
  | 2타임 | 14:30 ~ 15:00 |
  | 3타임 | 15:30 ~ 16:00 |
  | 4타임 | 16:30 ~ 17:00 |
- 안마사 1인 기준이므로 **각 슬롯은 1명만 확정 가능**
- 차주 예약 대상: 월~금 5일 × 4타임 = **주당 최대 20슬롯**

### 3.2 예약 주기 (주 단위 운영)
- **예약 오픈**: 매주 수요일 오전 9시
- **예약 마감**: 매주 수요일 오후 5시 (실제 신청 접수는 **16:59까지**, **17:00 정각 마감**)
- **신청 가능 시간대**: 매주 수요일 09:00 ~ 16:59 동안만 신청 가능
- **예약 대상 기간**: 차주(다음 주) 월요일 ~ 금요일 (5일)
- **운영 사이클**: 매주 수요일에만 다음 주 일정을 신청할 수 있음

> 예시
> - 6/10(수) 09:00 오픈 → 6/10(수) 17:00 마감(16:59까지 접수) → 대상 기간: 6/15(월)~6/19(금)
> - 6/17(수) 09:00 오픈 → 6/17(수) 17:00 마감(16:59까지 접수) → 대상 기간: 6/22(월)~6/26(금)

> 시간대별 예약 화면 상태
> - 수요일 09:00 이전: "예약 오픈 전" (신청 불가, 안내 문구 노출)
> - 수요일 09:00 ~ 16:59: "예약 가능" (신청·취소 화면 활성)
> - 수요일 17:00 ~ 다음날(목) 17:00: "탈락자 한정 재신청 가능" (대상자만 선착순 신청)
> - 목요일 17:00 이후 ~ 다음 수요일 09:00: "예약 마감" (신청 불가)

### 3.3 예약 상태 정의
| 상태 | 설명 |
|------|------|
| 신청(요청) | 회원이 날짜·시간을 선택해 접수한 상태 (일반 신청, 확정 대기) |
| 확정(완료) | 예약이 최종 확정된 상태 (관리자 확정 또는 재신청 즉시 확정), 완료 메일 발송 |
| 탈락 | 우선권에서 밀려나 확정되지 못한 신청 (재신청 안내 메일 발송 대상) |
| 취소 | 마감 전 신청자가 취소했거나 관리자가 취소한 신청 |

### 3.4 우선권 (Priority) 규칙
- **목적**: 동일 날짜·시간 슬롯에 여러 명이 신청했을 때 공정한 배분
- **1순위 기준**: **마지막으로 안마서비스를 받은 날짜(예약 확정 완료 기준)가 가장 오래된 사용자**
  - 한 번도 받지 않은 사용자는 최우선권으로 간주
- **2순위 기준(동률 시)**: 마지막 이용일이 동일한 경우 **신청 시각이 가장 오래된(먼저 신청한) 사용자**에게 우선권
- **서비스 첫 운영 시**: 모든 사용자가 이용 이력이 없으므로, 신청자 상세 정보에 **"이력 없음"**으로 표시되며, 이 경우 우선권 자동 판단이 불가하므로 **관리자가 수동으로 예약 완료 처리**한다.
- 관리자 페이지에서 각 신청자의 **마지막 이용일**, **신청 시각**, **우선권 보유 여부(O/X)**를 함께 표시

### 3.5 일반 신청 취소 규칙
- 일반 신청은 **마감(수요일 17:00) 전까지(=16:59까지) 취소 가능**
- 마감(17:00) 이후에는 **취소 불가**
- 취소된 슬롯은 다시 신청 가능한 빈 슬롯으로 전환
- **회원의 취소는 우선권 산정(마지막 이용일)에 영향을 주지 않는다.**

### 3.6 탈락자 재신청 규칙 (선착순 · 즉시 확정 · 취소 불가)
- **대상**: 동일 슬롯 중복 신청에서 우선권에 밀려 탈락한 신청자
- **재신청 가능 시간**: 마감(수요일 17:00) 직후부터 **다음날(목요일) 오후 5시까지**
- **재신청 대상 슬롯**: 마감 시점 기준 **비어있는(미확정) 날짜·시간 슬롯**
- **처리 방식**: **선착순** — 먼저 신청한 사람이 해당 슬롯을 차지
- **즉시 확정**: 관리자 확정 절차 없이 **신청하는 즉시 예약 확정** 처리 → 확정 메일 자동 발송
- **취소 불가**: 재신청은 즉시 확정되므로 **취소할 수 없다.**
- **동시 신청 충돌 처리**: 두 명 이상이 동시에 같은 슬롯을 신청할 경우, 신청 시점에 이미 확정된 슬롯이면 **"이미 예약이 완료된 날짜 및 시간대입니다."** 알럿을 표시하고 신청을 거절한다.
- 재신청 창 동안에는 **탈락자만** 신청 화면에 접근 가능

### 3.7 안마사 휴가 규칙
- 관리자는 **예약 오픈(수요일 09:00) 전에 미리** 휴가 날짜를 등록한다.
- **예약 오픈 후에는 휴가 등록 불가** (해당 주차 일정에 대해 추가 휴가 등록 차단)
- 휴가 지정 날짜는 회원 캘린더에서 **예약 불가(비활성)**로 표시되며 예약을 받지 않는다.

## 4. 사용자 시나리오 (User Flow)

### 4.1 회원 - 회원가입 및 로그인
1. 비회원이 회원가입 페이지 진입
2. 기본 정보 입력(이름, 이메일, 비밀번호 등)
3. 입력한 이메일로 **인증 메일 발송**
4. 메일 내 인증 링크 클릭 → 인증 완료
5. 로그인 가능 상태로 전환

### 4.2 회원 - 예약 신청 (수요일 09:00~16:59)
1. 로그인 후 예약 캘린더 진입
2. 차주(월~금) 예약 가능 날짜/시간대 확인 (하루 4타임)
   - 휴가 날짜·신청 불가 슬롯·오픈 전/마감 상태는 비활성 표시
3. 원하는 날짜·시간 슬롯 선택
4. 예약 신청 제출
5. **신청 접수 완료** 안내 (※ 예약 확정 아님을 명확히 안내)
6. 마이페이지에서 신청 상태 확인 및 **마감 전 취소 가능**

### 4.3 마감 시점 자동 처리 (매주 수요일 17:00)
1. 일반 신청 마감 (16:59까지만 접수, 17:00 정각 마감, 이후 취소 불가)
2. 동일 날짜·시간 중복 신청 건에 대해 우선권 정렬 (마지막 이용일 → 동률 시 신청 시각)
3. 우선권에서 밀려난(탈락) 신청자에게 **재신청 안내 메일 발송**
   - 비어있는 날짜·시간대 정보 포함
   - 재신청 가능 시간(다음날 17:00까지)·선착순·즉시 확정·취소 불가 안내 포함
4. **탈락자 한정 재신청 창 오픈** (수 17:00 ~ 목 17:00)
5. 재신청 창 마감 후에는 모든 신청 불가, 다음 차주 일정은 **다음 주 수요일 오전 9시**에 신규 오픈

### 4.4 회원 - 탈락 후 재신청 (선착순 · 즉시 확정)
1. 탈락 안내 메일 수신 (비어있는 슬롯·재신청 가능 시간 포함)
2. 재신청 창 시간(수 17:00~목 17:00) 내 로그인 → 재신청 화면 진입
3. 비어있는 날짜·시간 슬롯 중 선택하여 재신청 제출
   - 이미 확정된 슬롯일 경우 **"이미 예약이 완료된 날짜 및 시간대입니다."** 알럿 표시 후 거절
4. **선착순 즉시 확정** → 예약 완료 메일 자동 발송 (이후 취소 불가)

### 4.5 관리자 - 예약 확정 처리 (일반 신청 대상)
1. 관리자 로그인
2. 날짜별 신청 현황 조회
3. 신청자 상세 정보 확인 (이름, 이메일, 마지막 이용일, 신청 시각, 우선권 여부 등)
   - 첫 운영 등 이력이 없는 경우 **"이력 없음"**으로 표시
4. 우선권 보유자 기준으로 **예약 확정** 처리 (이력 없음일 경우 관리자 수동 판단)
5. 확정 시 신청자에게 **예약 완료 메일 자동 발송**
6. (선택) 우선권에서 밀려난 신청자에게 **탈락/재신청 안내 메일 발송**

### 4.6 관리자 - 안마사 휴가 관리 (예약 오픈 전)
1. 휴가 관리 메뉴 진입 (예약 오픈 전에 수행)
2. 캘린더에서 차주 휴가 날짜 지정/해제
3. 지정된 날짜는 오픈 시 회원 예약 화면에서 예약 불가로 반영
4. 예약 오픈(수 09:00) 이후에는 해당 주차 휴가 등록 불가

| 메일 종류 | 발송 시점 | 수신자 | 주요 내용 |
|-----------|-----------|--------|-----------|

| 탈락/재신청 안내 메일 | 마감(수 17:00) 시 / 관리자 발송 | 우선권 탈락자 | 비어있는 날짜·시간, 재신청 가능 시간(다음날 17:00까지), 선착순·즉시 확정·취소 불가 안내 |

## 6. 관리자 페이지 주요 기능

### 6.1 예약 현황 관리
- 날짜별 신청 목록 조회 (일반 신청 / 재신청 구분 표시)
- 신청자 상세 정보 표시: 이름, 이메일, 신청 날짜·시간(슬롯), **신청 시각**, **마지막 이용일(또는 "이력 없음")**, **우선권 여부(O/X)**, 신청 상태
- 동일 날짜·시간 중복 신청자 그룹 표시 (우선권 정렬 결과 표시)
- 예약 확정 / 취소 처리 (이력 없음 시 수동 확정)
- 우선권 탈락자 대상 재신청 안내 메일 발송

### 6.2 안마사 휴가 관리
- 휴가 날짜 등록/수정/삭제 (예약 오픈 전에만 가능)
- 휴가 반영 결과 미리보기(예약 불가 처리 확인)

### 6.3 운영 설정 (확장 고려)
- 운영 시간/시간 슬롯 단위 설정 (기본: 13:30~17:00, 4타임)
- 예약 오픈/마감 요일·시간 설정 (현재 수요일 09:00 오픈 / 17:00 마감)
- 탈락자 재신청 창 시간 설정 (현재: 마감 후 ~ 다음날 17:00)
- 메일 템플릿 관리

│
├── [회원 영역]

│   │   └── 탈락자 재신청 (재신청 창 시간 한정, 선착순·즉시 확정·취소 불가)
│   ├── 마이페이지
│   │   ├── 내 예약 신청 내역
│   │   ├── 예약 상태 확인 (신청/확정/탈락/취소)

│   └── 알림(메일) 내역 (선택)

        ├── 시간 슬롯/운영 시간 설정
        ├── 오픈/마감/재신청 창 시간 설정
        └── 메일 템플릿 관리

## 10. 정책 확정 사항 (반영 완료)

1. **재신청 취소**: 즉시 확정 건이므로 **취소 불가**
2. **노쇼 처리**: 확정 후 노쇼에 대한 별도 처리 **없음**
3. **휴가 등록 시점**: **예약 오픈 전**에만 등록 가능, 오픈 후 등록 불가
4. **재신청 동시 클릭 처리**: 이미 확정된 슬롯이면 **"이미 예약이 완료된 날짜 및 시간대입니다."** 알럿 표시 후 거절
5. **첫 운영 시 이력 없는 사용자**: 상세 정보에 **"이력 없음"** 표시, 관리자가 **수동으로 예약 완료 처리**
6. **회원 취소의 우선권 영향**: **영향 없음** (마감 전 취소는 마지막 이용일에 반영되지 않음)
